## OQ-003: Провайдер TTS (озвучка «носителя»)

| | |
|---|---|
| **Вопрос** | Какой сервис TTS выбрать для production? |
| **Кандидаты** | Azure Neural TTS, Google Cloud TTS, Amazon Polly, ElevenLabs |
| **Критерий** | Neural voice, естественность, цена за 1M символов, языковое покрытие |
| **Допущение в docs** | Абстракция `TtsProvider`; MVP — один провайдер через BFF proxy |
| **Fallback** | Web Speech API с badge «синтез браузера» |
| **Статус** | Open |
