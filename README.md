# Click&Speak

Минималистичное веб-приложение для изучения иностранной лексики: **один клик — готовая карточка**, интервальное повторение (как Anki), колоды (как Quizlet), без лишней геймификации.

## Статус проекта

- **Код приложения:** не начат  
- **UI-мокапы:** [`stitch_linguo_vocab_flashcards/`](stitch_linguo_vocab_flashcards/)  
- **Документация:** [`docs/`](docs/) — источник правды для разработки и приёмки  

## Быстрый старт для команды

1. Прочитать [Product Vision & PRD](docs/01-product-vision-prd.md) — scope MVP.  
2. Открыть [UX/UI Spec](docs/02-ux-ui-spec.md) и мокапы в браузере.  
3. Для разработки: [Architecture](docs/03-architecture.md) → [Data Model](docs/04-data-model.md) → [API Spec](docs/05-api-spec.md).  
4. Для QA: [Acceptance Criteria](docs/07-acceptance-criteria.md).  

## Ключевые возможности (MVP)

- Quick Add: ввод слова → автоматический перевод, пример, часть речи, озвучка  
- Колоды: создание, группировка, поиск, импорт CSV  
- Обучение: flip-карточки, SRS (Again / Hard / Good / Easy), shuffle  
- Статистика: streak, активность, прогресс без перегруза  
- Адаптивный web (desktop + mobile)
