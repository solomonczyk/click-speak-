# Product Vision & PRD — Click&Speak

**Версия:** 1.0.0  
**Дата:** 2026-05-23

---

## 1. Executive Summary

**Click&Speak** — веб-приложение для самостоятельного изучения иностранной лексики. Пользователь вводит одно слово или фразу и получает готовую учебную карточку (перевод, пример в контексте, метаданные, озвучка носителя). Карточки объединяются в колоды, повторяются по алгоритму интервального повторения (SRS), порядок в сессии можно перемешивать.

Продукт сознательно **не** копирует социальные и игровые механики конкурентов: фокус на скорости ввода, качестве аудио и спокойном UI для длительных занятий.

---

## 2. Проблема и решение

### Проблема

| Боль | Как проявляется |
|------|-----------------|
| Медленное создание карточек | В Anki/Quizlet много полей и ручной работы |
| Слабая озвучка | TTS роботизированный или отсутствует |
| Перегруз фичами | MemoWord/ReWord/другие — лишние экраны, квесты, подписки на всё |
| Разрозненность | Quizlet — колоды без сильного SRS; Anki — SRS без «одного клика» |

### Решение

Один главный цикл: **увидел слово → ввёл → выучил → увидел прогресс**. Всё остальное — вспомогательное или отложено.

## 3. Позиционирование и конкуренты

### Позиционирование

> «Quizlet по удобству колод + Anki по повторениям + MemoWord по скорости добавления — без мусора в интерфейсе.»

### Сравнение (высокий уровень)

| Продукт | Сильная сторона | Слабость для нашей ЦА | Click&Speak |
|---------|-----------------|------------------------|-------------|
| Quizlet | Колоды, простота | Слабый SRS | Сильный SRS |
| Anki | SRS, гибкость | Сложное создание карточек | Quick Add |
| MemoWord / ReWord | Быстрый ввод | Лишние фичи, шум UI | Минимализм |
| Duolingo | Мотивация | Не словарь пользователя | Свои слова |

### Уникальное ценностное предложение (UVP)

1. **One-click card** — одно поле ввода → полная карточка за секунды.  
2. **Native-quality audio** — приоритет neural TTS, кэш для повторного прослушивания.  
3. **Calm UI** — дизайн-система Linguistic Clarity: фокус, без квестов и XP.  

---

## 4. Целевая аудитория

### Primary persona: «Самостоятельный читатель»

- Возраст 18–45, уровень B1–C2 по изучаемому языку.  
- Читает книги/смотрит контент на иностранном, записывает слова.  
- Устройства: смартфон (дорога) + ноутбук (дом).  
- Готов платить за качество позже; MVP — бесплатный local-first.

### Anti-persona (не целимся в v1)

- Нулевой уровень «с нуля» без собственного словаря.  
- Школы/классы с LMS и домашними заданиями от учителя.  
- Пользователи, которым нужен только готовый курс (как Duolingo).

|---|--------|-------|--------|
| JTBD-1 | Встретил незнакомое слово | Быстро добавить его в колоду | Не потерять контекст и не тратить 5 минут на оформление |

| JTBD-5 | Хочу понять, не забрасываю ли | Видеть streak и простую статистику | Поддерживать привычку без стресса |

| ID | Требование | Приоритет |

| FR-D-01 | Создание колоды: название, описание, язык изучения (`sourceLang`), язык перевода (`targetLang`) | Must |
| FR-D-02 | Редактирование и удаление колоды (с подтверждением при удалении) | Must |

## 8. Out of Scope (MVP) — явный запрет на приёмку v1

Следующее **не разрабатывается** и **не тестируется** в MVP:

| Категория | Исключено |
|-----------|-----------|
| Социальное | Шаринг колод, друзья, публичные колоды, комментарии |
| Геймификация | XP, уровни, бейджи («Linguistic Elite»), множители streak, лидерборды |
| Контент | Word of the Day, AI-генерация целых колод, готовые курсы |
| Навыки кроме лексики | Grammar / Listening / Speaking breakdown в статистике |
| Уведомления | Push, email, in-app notification center |
| Монетизация | Pro-подписка, paywall (кроме заготовки в UI не показывать) |
| Header chrome | Иконки notifications/history без функции (скрыть или убрать) |
| Отчёты | Download Report PDF |
| Нативные приложения | iOS/Android store apps |
| Синхронизация | Мультиустройство в реальном времени |
| Офлайн enrichment | Quick Add без сети (только replay кэшированного audio) |

Элементы из мокапов, попадающие в этот список, перечислены в [02-ux-ui-spec.md](./02-ux-ui-spec.md).

## 10. Допущения и ограничения

| Тема | Решение для MVP |
|------|-----------------|
| Auth | Local-first, IndexedDB (Dexie), см. [03-architecture.md](./03-architecture.md) |

## 11. Метрики успеха MVP

### Продуктовые

| Метрика | Target |
|---------|--------|
| Quick Add latency (p50) | ≤ 3 с до playable audio (online) |
| Session completion rate | ≥ 70% сессий до конца очереди или осознанного exit |
| D7 retention (пилот) | ≥ 25% (при наличии analytics) |

### Технические

| Метрика | Target |
|---------|--------|
| Lighthouse Performance (mobile) | ≥ 85 |
| Потеря SRS после refresh mid-session | 0 |
| Critical bugs P0 | 0 на релизе |

### Качественные

- 5 пользователей из ЦА подтверждают: «быстрее, чем Anki, спокойнее, чем MemoWord».  
- Дизайн-ревью: соответствие [design-system.md](./design-system.md).
